# PROG6212_POE_Part2

## Description

This project is a C# WPF application developed for the PROG6212 POE (Part 2) assignment. It provides a user interface for managing lecturer and program coordinator/academic manager accounts, handling claims, and other related functionalities.

## Installation

1.  Clone the repository to your local machine:

    ```shell
    git clone <repository_url>
    ```

2.  Open the `PROG6212_POE.sln` file in Visual Studio.
3.  Ensure you have the required .NET SDK installed.
4.  Build the solution to restore NuGet packages and compile the application.

## Usage

1.  Run the application from Visual Studio.
2.  The main window will appear, allowing you to sign in as a lecturer or program coordinator/academic manager, or create a new account.
3.  Use the provided credentials to log in or create a new account.
4.  Depending on your account type, you will be directed to the corresponding dashboard.
